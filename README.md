# Share-File (Node AirDrop)

Repositorio basado en el proyecto "Share-File" (Node-AirDrop).

## Requisitos
- Node.js v14 o superior
- npm

## Instalación
1. Clonar el repositorio o descargar los archivos.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:

```
npm install
```

## Ejecutar

Para iniciar el servidor:

```
node server.js
```

El servidor por defecto escucha en el puerto definido en `server.js` (revisa ese archivo). Luego abre `index.html` en el navegador o usa la interfaz que provea el servidor.

## Notas
- Si existe un script `start` en `package.json`, también puedes usar `npm start`.
- Si al hacer `git push` necesitas credenciales, configúralas en tu entorno (GitHub CLI, token o credenciales remotas).
